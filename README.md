# 🏋️ PowerHouse - Gym Management Frontend

PowerHouse is a modern and responsive gym management frontend application designed to help fitness centers provide a seamless digital experience for members.

The platform offers an intuitive interface for exploring fitness programs, trainers, memberships, schedules, and gym services.

## 📌 Overview

PowerHouse is built to create a strong online presence for fitness businesses while delivering an engaging user experience.

The application focuses on:

- Modern Gym Branding
- Membership Promotion
- Trainer Showcases
- Workout Programs

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/powerhouse.git
```

### Navigate to Project

```bash
cd powerhouse
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Application runs at:

```text
http://localhost:5173
```
